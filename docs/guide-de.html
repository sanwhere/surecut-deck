Mach aus einem übrigen Tablet ein Bedienpult für deinen PC

Surecut Deck startet einen kleinen Server auf deinem Windows-Rechner und liefert seine Oberfläche selbst aus. Das Tablet öffnet nur eine Webadresse: keine App zum Installieren, kein Konto, keine Cloud dazwischen.

So funktioniert es

Dein PC hostet esEine kleine Anwendung im Infobereich startet den lokalen Server und hält ihn am Laufen. Sie ist das Einzige, was du installierst.
Das Tablet öffnet eine AdresseJeder Browser im selben Netzwerk. Zum Startbildschirm hinzugefügt verhält es sich wie eine App.
Schaltflächen senden echte EingabenAus Berührungen werden echte Windows-Tastatur- und Mausereignisse, deshalb reagiert jedes Programm darauf.

Drei Schritte, einmalig.

Verbinden

1

Server starten

Führe start.cmd aus. Im Infobereich erscheint ein Symbol und bleibt dort. Klicke es jederzeit an, um die Kopplungsdaten zu sehen oder den Editor zu öffnen.

2

Adresse auf dem Tablet öffnen

Das Serverfenster listet alle Adressen auf, unter denen dein PC erreichbar ist. Gib eine davon im Browser des Tablets ein und danach den daneben angezeigten Kopplungscode.

3

Wie eine App wirken lassen

Tippe auf ⛶ für Vollbild oder nutze Zum Startbildschirm hinzufügen deines Browsers. Im Vollbild bleibt das Tablet außerdem wach, solange das Pult geöffnet ist.

Grundlagen

Schaltflächen können mehr als Tasten senden

Tippe auf den Stift für den Bearbeitungsmodus und dann auf eine Schaltfläche, um sie zu ändern, oder halte sie gedrückt, um sie an eine andere Stelle zu ziehen. Jede Schaltfläche hat eine Beschriftung, ein Symbol, eine Farbe und eine Aktion.

Eine Schrittfolge steht Zeile für Zeile da, ein Schritt pro Zeile: key ctrl+s, text Hello, delay 500, launch notepad, url https://…, cmd …. Die Schaltfläche Testen im Editor führt die Aktion einmal aus, ohne sie zu speichern; so bekommst du ein kniffliges Kürzel hin, bevor du dich darauf festlegst.

Symbole

Emoji wählen, oder das echte App-Symbol nehmen

Der Editor hat ein Symbolraster, du musst also nie ein Emoji von Hand tippen. Für Programme gibt es etwas Besseres: Zieh die Verknüpfung vom Desktop herein, dann kommt das echte Programmsymbol gleich mit.

Desktop

Zieh eine Verknüpfung vom Desktop herein

Öffne den Editor über das Symbol im Infobereich: Es erscheint ein richtiges Desktop-Fenster mit derselben Oberfläche darin. Lege eine Verknüpfung, ein Programm, einen Ordner oder eine Datei auf den Streifen oben: Das Ziel wird aufgelöst, das Programmsymbol extrahiert, und die Schaltfläche erscheint sofort auf dem Tablet.

Der Editor ist aus einem bestimmten Grund ein natives Fenster und kein Browser-Tab. Browser geben einer Webseite niemals den Pfad einer abgelegten Datei, und Chromium löst eine abgelegte Verknüpfung zusätzlich zu ihrem Ziel auf, ein Browser würde also versuchen, das ganze Programm zu lesen, statt zu erfahren, wo es liegt.

Abgleich

Von beiden Seiten bearbeiten, sofort

Tablet und Desktop-Editor sind dasselbe Pult, zweimal gesehen. Füge am PC eine Schaltfläche hinzu, und sie erscheint ohne Neuladen auf dem Tablet; benenne am Tablet eine Seite um, und die Seitenliste im Editor zieht mit. Jede Änderung wird sofort auf die Festplatte geschrieben, und die letzten dreißig Fassungen bleiben erhalten; eine Bearbeitung, die gleichzeitig auf zwei Geräten passiert, wird deshalb abgelehnt, statt die andere stillschweigend zu überschreiben.

Platz

Wische zwischen den Seiten

Fasse Schaltflächen zu Seiten zusammen: eine zum Bearbeiten, eine für Spiele, eine für das Meeting, in dem du immer sitzt. Wische an beliebiger Stelle auf dem Pult nach links oder rechts, um zu wechseln; der Seitenname erscheint kurz, damit du weißt, wo du gelandet bist. Im Bearbeitungsmodus sortiert dieselbe Geste stattdessen die Schaltflächen, die beiden kommen sich also nie in die Quere.

Platz

Seiten, Spalten und die schwebende Schaltfläche

Im Bearbeitungsmodus legt die untere Leiste Seiten an und entfernt sie und bestimmt, wie viele Spalten das Raster nutzt. Tippe eine Seitenlasche erneut an, um sie umzubenennen. Die schwebende Schaltfläche führt dich auch bei ausgeblendeten Leisten zu den Seiten, zum Editor und zu den Designs, und du kannst sie hinziehen, wo sie dir passt, denn sie ist das Einzige, was immer auf dem Bildschirm bleibt.

Aussehen

Elf Designs, und die Schaltflächen ziehen mit

Die meisten Designs stammen aus bekannten Farbschemata für Entwickler (Nord, Gruvbox, Solarized, Catppuccin Mocha, Rosé Pine), dazu zwei bewusst zweifarbige Terminal-Looks sowie ein neutrales Hell und Dunkel. Beim Wechsel werden deine Schaltflächenfarben unter Beibehaltung ihrer Farbtöne in die neue Palette übertragen: Aus einer blauen Schaltfläche wird das Blau des neuen Designs, und dein Aufbau liest sich weiterhin gleich.

Wenn ein Pult am Ende einfarbig ist, verteilt Design-Palette verteilen die Farben des Designs auf die Schaltflächen; Alle in Akzentfarbe färben setzt sie wieder auf eine einzige zurück. Jede Schaltfläche kann außerdem aus der Farbauswahl eine eigene Farbe bekommen, und die Akzentfarbe für die Leisten und die schwebende Schaltfläche wählst du selbst. Die Schaltflächenbeschriftung wechselt von allein zwischen Schwarz und Weiß, welche Farbe du auch triffst.

Bildschirm

Nutze den ganzen Bildschirm

Schaltflächen über den ganzen Bildschirm strecken lässt die Zeilen die volle Höhe teilen: Zwei Reihen Schaltflächen füllen ihn dann aus, statt unten eine Lücke zu lassen. Obere und untere Leiste automatisch ausblenden entfernt sie ganz und lässt nur die Schaltflächen stehen; die schwebende Schaltfläche holt sie zurück, oder halte eine leere Fläche drei Sekunden gedrückt.

Das Tablet ist außerdem so eingestellt, dass es wach bleibt: Der Bildschirm schaltet sich nicht ab, solange das Pult geöffnet ist.

Sprache

Zwanzig Sprachen, automatisch gewählt

Die Oberfläche gibt es auf Englisch, Türkisch, Spanisch, Französisch, Deutsch, Italienisch, Portugiesisch, Russisch, Ukrainisch, Polnisch, Arabisch, Persisch, Urdu, Chinesisch, Japanisch, Koreanisch, Hindi, Bengalisch, Indonesisch und Vietnamesisch. Sie folgt der Browsersprache, solange du keine wählst; Arabisch, Persisch und Urdu stellen das Layout auf rechts-nach-links um. Die Wahl gilt pro Gerät, Tablet und Desktop-Editor können sich also unterscheiden.

Gut zu wissen

Firewall
Windows fragt beim ersten Mal eventuell, ob die Verbindung erlaubt werden soll. Erreicht das Tablet den Server nicht, erlaube sie für private Netzwerke.
VPN
Manche VPN-Clients blockieren den lokalen Netzwerkverkehr. Verbindet sich nichts, probiere es mit ausgeschaltetem VPN.
Administratorfenster
Ein als Administrator laufendes Programm nimmt keine Eingaben von einem normalen Programm an. Starte den Server als Administrator, wenn deine Kürzel solche Fenster erreichen sollen.
Kopplungscode
Wer in deinem Netzwerk den Code hat, kann den PC steuern, auch Befehle ausführen. Behalte ihn für dich und hoste nicht in Netzwerken, denen du nicht traust.

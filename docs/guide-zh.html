把闲置平板变成电脑的控制台

Surecut Deck 在你的 Windows 电脑上运行一个小服务器，并自行提供界面。平板只需打开一个网址，不用安装应用，不用注册账号，中间也没有云。

工作方式

由你的电脑托管托盘里的小程序启动本地服务器并保持运行。你需要安装的只有它。
平板打开一个网址同一网络下的任意浏览器。添加到主屏幕后，用起来就像一个应用。
按钮发送真实输入按下会变成真正的 Windows 键盘和鼠标事件，因此任何程序都会响应。

三步，只需一次。

连接

1

启动服务器

运行 start.cmd。系统托盘会出现一个图标并常驻。随时点击它即可查看配对信息或打开编辑器。

2

在平板上打开地址

服务器窗口列出了所有可以访问到你电脑的地址。在平板浏览器里输入其中一个，然后填入旁边显示的配对码。

3

让它像个应用

点 ⛶ 进入全屏，或使用浏览器的添加到主屏幕。全屏时，只要控制台开着，平板也不会休眠。

基础

按钮不只是发送按键

点铅笔进入编辑模式，然后点任意按钮进行编辑，或者长按把它拖到别处。每个按钮都有标签、图标、颜色和一个动作。

步骤序列每行写一步：key ctrl+s、text Hello、delay 500、launch notepad、url https://…、cmd …。编辑器里的测试按钮会执行一次动作但不保存，因此你可以先把难调的快捷键试对，再决定保存。

打开应用 / 文件程序、文档或文件夹

图标

挑一个表情符号，或直接用应用的真实图标

编辑器里有图标网格，不必手动输入表情符号。对程序来说还有更好的办法：把桌面上的快捷方式拖进来，应用的真实图标会一并带过来。

桌面

从桌面拖入快捷方式

从托盘图标打开编辑器，会出现一个真正的桌面窗口，里面是同一套界面。把快捷方式、程序、文件夹或文件拖到顶部的条上：目标会被解析，应用图标会被提取，按钮随即出现在平板上。

编辑器做成原生窗口而非浏览器标签页，有其具体原因。浏览器从不把拖入文件的路径交给网页，而且 Chromium 还会把拖入的快捷方式解析成它的目标，于是浏览器会试图读取整个程序，而不是知道它在哪里。

同步

两边都能改，实时生效

平板和桌面编辑器是同一个控制台的两个视图。在电脑上加一个按钮，平板不用刷新就会出现；在平板上给页面改名，编辑器的页面列表也会跟着变。每次改动都会立即写入磁盘，并保留最近三十个版本。所以两台设备同时改动时，其中一次会被拒绝，而不是悄悄覆盖掉另一次。

指针

屏幕变成触控板

给按钮设置鼠标动作，按下后整块平板就成了触控板。两个滑块分别调节指针和滚动速度，滚动方向也可以反转，以符合你的习惯。

空间

滑动切换页面

把按钮分组到不同页面：一页用于剪辑，一页用于游戏，一页留给那个你总在里面的会议。在控制台上任意位置左右滑动即可在页面间切换；页面名会短暂显示，让你知道自己落在了哪一页。在编辑模式下，同样的手势用来调整按钮顺序，两者不会冲突。

空间

页面、列数和悬浮按钮

在编辑模式下，底部工具栏可以增删页面，并设置网格使用多少列。再点一次页面标签即可给它重命名。即使工具栏已隐藏，悬浮按钮依然能带你去页面、编辑器和主题，而且可以拖到任何你顺手的位置，因为它是唯一始终显示在屏幕上的东西。

外观

十一款主题，按钮也跟着变

多数主题取自开发者熟悉的配色方案，比如 Nord、Gruvbox、Solarized、Catppuccin Mocha、Rosé Pine；另外还有两款刻意只用双色的终端风格，以及中性的浅色与深色。切换主题时，按钮颜色会在保留色相的前提下转换到新配色：蓝色按钮变成新主题的蓝色，你的布局读起来依然一致。

如果整个控制台最后变成了一种颜色，按主题配色分配会把主题的颜色分散到各个按钮上；全部涂成强调色则把它们全都改回同一种。任意按钮也可以从取色器里选一个你自己的颜色，工具栏和悬浮按钮使用的强调色同样由你决定。无论你选了什么颜色，按钮文字都会自动在黑白之间切换。

屏幕

把整块屏幕用起来

拉伸按钮填满屏幕让各行平分全部高度：两行按钮会铺满屏幕，而不是在下方留出空白。自动隐藏上下工具栏则把它们彻底移除，只留下按钮；想让它们回来，可以点悬浮按钮，或者长按空白处三秒。

平板也已设置为保持唤醒：只要控制台开着，屏幕就不会自动熄灭。

语言

二十种语言，自动选择

界面提供英语、土耳其语、西班牙语、法语、德语、意大利语、葡萄牙语、俄语、乌克兰语、波兰语、阿拉伯语、波斯语、乌尔都语、中文、日语、韩语、印地语、孟加拉语、印尼语和越南语。在你未指定语言前，它会跟随浏览器语言；阿拉伯语、波斯语和乌尔都语会把布局切换为从右到左。语言按设备分别保存，因此平板和桌面编辑器可以不同。

值得了解

防火墙
Windows 首次可能会询问是否允许连接。若平板连不上服务器，请为专用网络放行。
VPN
部分 VPN 客户端会拦截局域网流量。如果怎么都连不上，试着关闭 VPN。
管理员窗口
以管理员身份运行的程序不接受普通程序的输入。若需要快捷键作用于这类窗口，请以管理员身份运行服务器。
配对码
网络中任何拿到配对码的人都能控制这台电脑，包括执行命令。请自己保管好，不要在不信任的网络上运行。

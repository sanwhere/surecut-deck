自前ホスト · アプリストア不要 · データはネットワークの外に出ません

余っているタブレットをパソコンの操作卓にする

Surecut Deck は Windows パソコン上で小さなサーバーを動かし、自分の画面を自分で配信します。タブレットはウェブアドレスを開くだけ——インストールするアプリも、作るアカウントも、あいだに入るクラウドもありません。

しくみ

ホストはあなたのパソコン通知領域の小さなアプリがローカルサーバーを起動し、動かし続けます。入れるのはこれだけです。
タブレットはアドレスを開くだけ同じネットワーク上のどのブラウザーでも。ホーム画面に追加すればアプリのように使えます。
ボタンは本物の入力を送る押すと本物の Windows キーボード・マウスイベントになるので、どのプログラムも反応します。

三つの手順、最初の一度だけ。

接続する

1

サーバーを起動する

start.cmd を実行します。通知領域にアイコンが現れ、そのまま常駐します。いつでもクリックすればペアリング情報を見たり、エディターを開いたりできます。

2

タブレットでアドレスを開く

サーバーの画面には、パソコンに到達できるアドレスがすべて並びます。そのひとつをタブレットのブラウザーに入力し、隣に表示されたペアリングコードを入れてください。

3

アプリらしくする

⛶ で全画面にするか、ブラウザーのホーム画面に追加を使ってください。全画面ではデッキを開いているあいだタブレットもスリープしません。

基本

ボタンはキーを送るだけではありません

鉛筆を押して編集モードに入り、ボタンをタップして編集します——長押しすれば別の場所へドラッグできます。ボタンにはラベル、アイコン、色、そして動作がひとつずつあります。

動作内容
キーボードショートカットどんな組み合わせでも——ctrl+c、win+d、alt+f4、メディアキー、音量
テキストを入力改行も含めて、決まった文章を代わりに書きます
アプリ / ファイルを開くプログラム、書類、フォルダー
ウェブアドレスを開く既定のブラウザーでリンクを開きます
コマンドを実行PowerShell または CMD
マウスタブレットの画面をタッチパッドに変えます

アイコン

絵文字を選ぶか、アプリ本来のアイコンを使う

エディターにはアイコンの一覧があるので、絵文字を手で入力する必要はありません。プログラムにはもっと良い方法があります。デスクトップからショートカットをドラッグすれば、アプリ本来のアイコンも一緒に付いてきます。

デスクトップ

デスクトップからショートカットをドラッグする

通知領域のアイコンからエディターを開くと、同じ画面を内側に持つ本物のデスクトップウィンドウが現れます。上部の帯にショートカット、プログラム、フォルダー、ファイルを落としてください。対象が解決され、アプリのアイコンが取り出され、ボタンがすぐタブレットに現れます。

エディターがブラウザーのタブではなくネイティブウィンドウなのには理由があります。ブラウザーは落とされたファイルのパスをウェブページに渡しません。さらに Chromium は落とされたショートカットをその対象に解決してしまうため、ブラウザーは場所を知る代わりにプログラム全体を読もうとしてしまいます。

ポインター

画面がタッチパッドになる

ボタンにマウスの動作を割り当てると、押した瞬間からタブレット全体がトラックパッドになります。二つのスライダーでポインターとスクロールの速さを決められ、スクロール方向は好みに合わせて逆にできます。

広さ

必要なだけページを作れます

ボタンをページごとにまとめましょう——編集用に一枚、ゲーム用に一枚、いつも入っているあの会議用に一枚。デッキ上で左右にスワイプすれば切り替わります。バーを隠していても、フローティングボタンからページ、エディター、テーマに行けますし、使いやすい場所へドラッグして構いません。

見た目

十一のテーマ、ボタンもそれに合わせて変わる

多くは開発者になじみのある配色から取っています——Nord、Gruvbox、Solarized、Catppuccin Mocha、Rosé Pine——加えて、あえて二色だけの端末風が二つと、中立的な明るいテーマと暗いテーマ。テーマを変えると、ボタンの色は色相を保ったまま新しい配色へ変換されます。青いボタンは新しいテーマの青になり、並びの読み取りやすさはそのままです。

画面

画面をすべて使う

画面いっぱいに広げるを使うと行が高さを分け合い、二行のボタンでも下に余白を残さず画面を埋めます。バーを自動的に隠すは上下のバーを完全に取り除き、ボタンだけを残します——フローティングボタンで戻せますし、空いている場所を三秒長押ししても戻ります。

タブレットはスリープしない設定にもしてあります。デッキを開いているあいだ画面は消えません。

言語

二十言語、自動で選ばれます

画面は英語、トルコ語、スペイン語、フランス語、ドイツ語、イタリア語、ポルトガル語、ロシア語、ウクライナ語、ポーランド語、アラビア語、ペルシア語、ウルドゥー語、中国語、日本語、韓国語、ヒンディー語、ベンガル語、インドネシア語、ベトナム語で用意されています。自分で選ばないかぎりブラウザーの言語に従い、アラビア語・ペルシア語・ウルドゥー語では右から左のレイアウトに切り替わります。設定は端末ごとなので、タブレットとデスクトップのエディターで別々にできます。

知っておくとよいこと

ファイアウォール
初回に Windows が接続の許可を求めることがあります。タブレットからサーバーに届かない場合は、プライベートネットワークで許可してください。
VPN
VPN クライアントによってはローカルネットワークの通信を遮断します。何も接続できないときは VPN を切って試してください。
管理者権限のウィンドウ
管理者として動いているプログラムは、通常のプログラムからの入力を受け付けません。そうしたウィンドウにショートカットを届けたい場合は、サーバーを管理者として実行してください。
ペアリングコード
ネットワーク上でコードを知っている人は、コマンドの実行を含めてパソコンを操作できます。コードは他人に渡さず、信頼できないネットワークでは動かさないでください。
